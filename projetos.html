Projeto de Analise do Censo de 2022

Objetivo do Projeto:

Analisar os dados do Censo Demográfico de 2022 para compreender as transformações socioeconômicas e demográficas do município de São José dos Campos, com investigação especial no crescimento e no perfil da Zona Oeste, a fim de subsidiar a necessidade de investimentos em equipamentos públicos, especificamente creches e centros de atividades para idosos.

Tecnologias utilizadas:

Python
Html
CSS
Pandas
Flask
Git e Github
Bootstrap
AWS
