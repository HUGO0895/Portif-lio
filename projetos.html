Projeto de Analise do Censo de 2022:

Objetivo do Projeto:

Analisar os dados do Censo Demográfico de 2022 para compreender as transformações socioeconômicas e demográficas do município de São José dos Campos, com investigação especial no crescimento e no perfil da Zona Oeste, a fim de subsidiar a necessidade de investimentos em equipamentos públicos, especificamente creches e centros de atividades para idosos.

Tecnologias utilizadas:

Python
Html
CSS
Pandas
Flask
Git e Github
Bootstrap
AWS

Minha Participação

Neste projeto,atuei como Product Owener (PO) e também contribuí ativamente nas atividades técnicas.Minhas principais entregas incluíram a definição de metas junto á equipe,a participação direta na raspagem de dados do Censo, a contribuição no desenvolvimento do back-end do site e a elabolação da documentação no github.Demonstrei agilidade na adaptação a mudanças de funções durante o projeto.Atualmente a o planejamento para o final do projeto está em andamento.
